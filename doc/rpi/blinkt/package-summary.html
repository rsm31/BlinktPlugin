Package rpi.blinkt

Event summary
BlinktHelperHelper event for the Blinkt Plugin, to control a Pimoroni Blinkt! APA102C LED board from Apama EPL. Requires the Blinkt Plugin to be installed. It is recommended to always use the helper event in your EPL code rather than invoking the plugin directly.

 

 

 
Monitor summary
BlinktSetupMonitor that exists only so that its onload() action can configure the Raspberry Pi GPIO pins for the Blinkt LED board.
